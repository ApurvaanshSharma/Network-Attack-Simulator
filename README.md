# 🛡️ Network Attack Simulator (Sandbox)

A Python-based cybersecurity training tool that simulates real-world network attacks in a safe, sandboxed environment.

## 🚀 Features

* 7 Attack Simulations:

  * Port Scan
  * Brute Force
  * DoS Attack
  * Reconnaissance
  * MITM Attack
  * SQL Injection
  * Lateral Movement
* CLI + Web GUI
* Real-time logging (SSE)
* Virtual network with firewall & IDS

## ⚠️ Disclaimer

This project is for **educational purposes only**.
No real network traffic is generated.

## ▶️ Run the Project

```bash
python main.py
```

## 🧠 Tech Stack

* Python
* HTTP Server
* Threading
* In-memory simulation
